# GIT BASIC

**1. Git System Initialization**
``` sh
$ git init
```
**2. Save into staging area (temporary changes)**
``` sh
$ git add
```
**3. Commit (permanent changes)**
``` sh
$ git commit [option] [argument]
```
&nbsp;&nbsp;option:
- ``-m`` : menambahkan pesan commit
``` sh
$ git commit -m "type: description"
```
